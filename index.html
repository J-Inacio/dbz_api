Sobre o Projeto

Este projeto foi desenvolvido como trabalho de conclusão da disciplina de Web Programming for Front-End.

O objetivo principal foi construir uma aplicação web capaz de consumir e exibir dados de uma API externa de personagens. Seguindo os requisitos técnicos propostos, a interface foi construída estritamente com HTML, CSS e JavaScript Vanilla, sendo a renderização dos cards e modais feita de forma 100% dinâmica através da manipulação do DOM.

Além de proporcionar ótimos desafios técnicos de lógica e assincronicidade, desenvolver essa aplicação foi um processo extremamente divertido, unindo a resolução de problemas reais de programação com a nostalgia de trabalhar com os personagens que marcaram a minha infância.
